# Consul

Consul service discovery

[consul.io](https://www.consul.io/){:target="_blank"}

[Consul on AWS](https://aws.amazon.com/quickstart/architecture/consul/){:target="_blank"}


## Commands

	-- Install consul
	$ brew install consul
	
	-- Start consul in dev mode
	$ consul agent -dev -node machine

	-- Stop consul
	$ consul leave
