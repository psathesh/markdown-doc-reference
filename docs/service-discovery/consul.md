# Consul

Consul service discovery

[consul.io](https://www.consul.io/)

[Consul on AWS](https://aws.amazon.com/quickstart/architecture/consul/)


## Commands

	-- Install consul
	$ brew install consul
	
	-- Start consul in dev mode
	$ consul agent -dev -node machine

	-- Stop consul
	$ consul leave
