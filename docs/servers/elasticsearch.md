## Installation

Download and unpack the Elasticsearch official distribution.

Run the below on Mac.

`bin/elasticsearch`

Open the below url on the browser.

`http://localhost:9200/`


## Commands

#### commands

`$ cd /Users/upurusa/tools/servers/elasticsearch/elasticsearch-7.8.0`

`$ ./bin/elasticsearch`
